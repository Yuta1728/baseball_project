# 野球選手成績予測システム（Webスクレイピング × 機械学習）

## 概要

本プロジェクトは、プロ野球選手の打撃成績データをWebスクレイピングによって取得し、機械学習（線形回帰）を用いて打率および本塁打数を予測するデータ分析アプリケーションです。

対象データを取得後、前処理・相関分析・特徴量選択を行い、チームごとの予測モデルを作成します。

本システムでは以下2球団を対象に分析を実施しています。

* 中日ドラゴンズ
* 読売ジャイアンツ

## 機能一覧

### 1. Webスクレイピング

指定サイトから野手成績データを取得します。

取得対象：

* 選手名
* チーム
* 打率
* 本塁打
* その他打撃指標

取得データは CSV として保存されます。

出力ファイル：

```bash
playerData.csv
```

---

### 2. データ前処理

取得したデータに対して以下を実施します。

* CSV読込
* 数値型変換
* 欠損値補完（NaN → 0）
* パーセント記号除去
* 文字列列の除外

対象外列：

```text
選手名
チーム
守備
```

---

### 3. 相関分析・特徴量選択

目的変数との相関係数を計算し、学習に利用する説明変数を自動選択します。

#### 打率モデル

```text
相関係数 >= 0
```

#### 本塁打モデル

```text
相関係数 >= 0.6
```

---

### 4. 機械学習による予測

アルゴリズム：

```text
Linear Regression（線形回帰）
```

学習対象：

| チーム     | 学習データ     |
| ------- | --------- |
| 中日予測モデル | 中日以外の全チーム |
| 巨人予測モデル | 巨人以外の全チーム |

処理内容：

1. 学習データ作成
2. 訓練・テスト分割
3. モデル学習
4. RMSE評価
5. 予測実行

評価指標：

```text
RMSE（Root Mean Squared Error）
```
